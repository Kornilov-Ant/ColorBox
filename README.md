Название проекта:
<br><b>ColorBox</b>
<br>
<br>Описание:
<br>Калькулятор просчета нужного материала для производства упаковки в промышленную типографию
<br>Приложение с визуализацией на SWING (для удобства взаимодействия сотрудников компании).
<br>
<br><b>Стек:</b>
<br>- Java 15
<br>- Maven
<br>- JUnit 5
<br>- Lombok
<br>- Swing
